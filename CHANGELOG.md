# 更新日志

本文档记录了ETF投资仪表盘的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-02-04

### 新增功能
- ✨ 多数据源ETF数据获取系统
  - 支持akshare数据源
  - 支持tushare数据源（含代理支持）
  - 自动故障转移机制
  - 智能重试和超时控制

- 📈 完整技术指标分析系统
  - MA5/20/30移动平均线计算
  - RSI14相对强弱指数
  - 最大回撤风险指标
  - 趋势状态自动识别

- 🎯 智能投资信号生成
  - 多因子信号模型
  - 趋势过滤机制
  - RSI超买超卖过滤
  - 回撤风险控制
  - 信号强度量化评分

- 💼 专业投资组合管理
  - 多ETF组合配置
  - 动态权重管理
  - 自动再平衡建议
  - 组合表现分析
  - 风险分散评估

- 🔧 健壮系统架构
  - 模块化组件设计
  - 系统集成器统一管理
  - 完善错误处理机制
  - 性能监控系统
  - 智能缓存机制

- 📱 现代化Web界面
  - 基于Streamlit的响应式设计
  - 多页面导航系统
  - 交互式图表展示
  - 实时数据更新
  - 用户友好的错误提示

- 🧪 完整测试套件
  - 单元测试覆盖
  - 集成测试验证
  - 性能测试监控
  - 数据验证测试

### 技术特性
- **数据获取**: 90天历史数据支持，确保技术指标计算准确性
- **缓存系统**: 24小时智能缓存，显著提升响应速度
- **错误处理**: 分级错误处理，自动恢复机制
- **配置管理**: JSON配置文件，支持运行时调整
- **日志系统**: 分级日志记录，便于问题诊断
- **性能监控**: 实时性能指标追踪

### 配置优化
- 技术指标参数优化（MA30替代MA60）
- 数据获取范围调整（30天→90天）
- 超时和重试参数优化
- 缓存策略改进

### 文档完善
- 详细的README文档
- 完整的API文档
- 开发指南和贡献指南
- 部署和配置说明
- 故障排除指南

### 已知问题修复
- 修复技术指标数据长度不足警告
- 修复性能监控格式错误
- 优化内存使用和性能
- 改进错误提示和用户体验
