# ETF投资仪表盘 v1.0.0 发布说明

## 🎉 首次发布

我们很高兴地宣布ETF投资仪表盘v1.0.0正式发布！这是一个功能完整、生产就绪的ETF投资分析工具。

## 📦 发布内容

### 核心功能
- ✅ **多数据源ETF数据获取**：支持akshare和tushare数据源
- ✅ **专业技术指标分析**：MA5/20/30、RSI14、最大回撤等
- ✅ **智能投资信号生成**：多因子模型，自动信号过滤
- ✅ **投资组合管理**：权重管理、再平衡建议、风险分析
- ✅ **实时性能监控**：系统健康检查、错误处理、性能追踪

### 技术特性
- ✅ **模块化架构**：清晰的组件分离，易于扩展
- ✅ **健壮错误处理**：完善的错误恢复机制
- ✅ **智能缓存系统**：24小时数据缓存，提升响应速度
- ✅ **灵活配置管理**：JSON配置文件，支持运行时调整
- ✅ **完整测试套件**：单元测试、集成测试、性能测试

### 用户界面
- ✅ **现代化Web界面**：基于Streamlit的响应式设计
- ✅ **多页面导航**：概览、ETF详情、组合管理、设置页面
- ✅ **交互式图表**：使用Plotly的专业金融图表
- ✅ **实时数据更新**：自动刷新和状态同步

## 🚀 快速开始

### 最简安装
```bash
git clone https://github.com/your-username/etf-dashboard.git
cd etf-dashboard
pip install -r requirements.txt
python start_app.py
```

### Docker部署
```bash
docker-compose up -d
```

## 📊 系统要求

- **Python**: 3.8+
- **内存**: 8GB+ (推荐)
- **存储**: 1GB+ 可用空间
- **网络**: 稳定的互联网连接

## 🔧 配置说明

### 数据源配置
支持多个数据源，具有自动故障转移功能：
- **akshare**: 免费，支持A股ETF数据
- **tushare**: 需要token，支持更丰富的数据

### 技术指标配置
可自定义的技术指标参数：
- 移动平均线周期：[5, 20, 30]
- RSI周期：14
- 超买/超卖阈值：70/30

## 🔄 升级路径

### 从开发版本升级
如果您之前使用过开发版本：
1. 备份配置文件：`config/settings.json`
2. 清理缓存：删除 `data/cache/` 目录
3. 更新代码：`git pull origin main`
4. 重新安装依赖：`pip install -r requirements.txt`
5. 重启应用：`python start_app.py`
